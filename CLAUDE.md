# FORGE

This project is the FORGE Claude Code plugin. Skills, agents, and hooks are auto-discovered from their respective directories.

## Quick Reference

- Start any task with `/forge:think`
- All skills are namespaced: `/forge:build`, `/forge:review`, `/forge:ship`, etc.
- See `docs/` for detailed documentation
- See `skills/forge/SKILL.md` for the full routing and workflow reference

## Project Conventions

When FORGE is installed into a project, respect any existing project conventions found in the project's own documentation. FORGE process augments — it does not override project-specific rules.
